O que é uma publicação web, quais os dois tipos de aplicações existentes e suas principais diferenças?

Publicação web é o processo de disponibilizar um conteúdo na internet, permitindo que qualquer usuário com acesso à web visualize esse conteúdo, seja ele uma página estática ou dinâmica. Esse conteúdo pode incluir textos, imagens, vídeos, aplicações interativas, entre outros.

Dois tipos de aplicações web:
Aplicações Web Estáticas: São páginas web cujo conteúdo não muda dinamicamente, ou seja, a página é carregada do servidor da mesma forma a cada vez que é acessada. Exemplo: um site institucional.
Aplicações Web Dinâmicas: O conteúdo dessas páginas pode mudar dependendo de interações do usuário, dados inseridos ou outras condições. Elas geralmente envolvem bancos de dados e são mais interativas. Exemplo: redes sociais e e-commerce.

O que é um domínio e um DNS?

Domínio: É o endereço único na internet que identifica um site ou serviço (exemplo: www.exemplo.com). Ele é registrado em uma entidade autorizada (registrar) e é usado para localizar o site na web.
DNS (Sistema de Nomes de Domínio): É um sistema que traduz os nomes de domínio (como www.exemplo.com) em endereços IP numéricos, necessários para localizar os servidores na rede. O DNS funciona como um catálogo telefônico da internet.

O que é um servidor compartilhado e um servidor dedicado, evidenciando suas vantagens e desvantagens?

Servidor Compartilhado: É um servidor onde múltiplos sites ou aplicações compartilham os recursos de um único servidor físico, como CPU, memória e espaço de armazenamento.
Vantagens: Custo mais baixo, pois os recursos são compartilhados. Facilidade de uso e manutenção, já que o provedor de hospedagem cuida do servidor.
Desvantagens: Desempenho pode ser afetado, pois os recursos são compartilhados com outros usuários. Menos controle sobre a configuração do servidor. Servidor Dedicado: É um servidor dedicado exclusivamente a um único cliente, oferecendo recursos totalmente exclusivos.
Vantagens: Maior desempenho e controle total sobre o servidor. Mais segurança e personalização.
Desvantagens: Custo mais alto. Exige conhecimentos técnicos para manutenção e configuração.

O que é um servidor web, explique os dois principais servidores. Como fazer o backup de um projeto de forma compactada? E como restaurá-lo?

Servidor Web: É um software que entrega conteúdo web aos usuários por meio de navegadores, usando o protocolo HTTP/HTTPS. Ele recebe solicitações de páginas e retorna os arquivos solicitados.
Dois principais servidores web: Apache HTTP Server:
Um dos servidores web mais populares, flexível e configurável. Ideal para sites de pequeno a grande porte.
Nginx: É conhecido por sua performance e por ser capaz de lidar com grandes volumes de tráfego, sendo frequentemente usado para servidores de alto desempenho.
Backup Compactado: Para fazer backup de um projeto (como um site), você pode compactar os arquivos em um arquivo .zip ou .tar.gz. No Linux, isso pode ser feito com o comando:
bash
Copiar código
tar -czvf backup_do_site.tar.gz /caminho/do/projeto
Restaurar o Backup: Para restaurar o backup, basta descompactar o arquivo:
bash
Copiar código
tar -xzvf backup_do_site.tar.gz

O que é um servidor de banco de dados, explique como fazer o backup e restaurar esse backup?

Servidor de Banco de Dados: É um servidor dedicado a armazenar e gerenciar dados para aplicativos e sites. Ele pode ser de tipos como MySQL, PostgreSQL, ou SQL Server.
Backup de Banco de Dados (MySQL): Para fazer o backup de um banco de dados MySQL, o comando seria:
bash
Copiar código
mysqldump -u usuario -p nome_do_banco > backup_banco.sql
Restaurar Backup: Para restaurar um backup do MySQL, o comando seria:
bash
Copiar código
mysql -u usuario -p nome_do_banco backup_banco.sql

O que é um serviço SSH, e como ele permite acessar um servidor web para transferir arquivos de uma máquina local para o servidor?

SSH (Secure Shell) é um protocolo que permite acessar remotamente servidores de forma segura. Ele é usado para administrar servidores, transferir arquivos e executar comandos.
Transferência de arquivos usando SCP (Secure Copy Protocol):
Para transferir arquivos de uma máquina local para um servidor remoto, o comando seria:
bash
Copiar código
scp arquivo_local usuario@ip_do_servidor:/caminho/do/destino

O que é o Linux e cite as principais distribuições usadas como servidor web?

Linux é um sistema operacional de código aberto, utilizado em uma vasta gama de dispositivos, incluindo servidores. Sua estabilidade, segurança e baixo custo são as razões principais para sua popularidade.
Principais distribuições Linux para servidores web:
Ubuntu Server: Muito popular por ser fácil de usar e com grande comunidade.
CentOS: Baseado no Red Hat, muito utilizado em servidores corporativos.
Debian: Reconhecido pela sua estabilidade.
Red Hat Enterprise Linux (RHEL): Usado em ambientes corporativos, pago e com suporte.

Explique alguns comandos básicos do Linux.

ls: Lista os arquivos e diretórios no diretório atual.
cd: Muda o diretório atual. Exemplo: cd /home/user.
mkdir: Cria um novo diretório. Exemplo: mkdir novo_diretorio.
rm: Remove arquivos ou diretórios. Exemplo: rm arquivo.txt.
cp: Copia arquivos ou diretórios. Exemplo: cp arquivo1.txt arquivo2.txt.
mv: Move ou renomeia arquivos. Exemplo: mv arquivo.txt novo_arquivo.txt.
top: Exibe o uso de recursos do sistema em tempo real.
chmod: Altera permissões de arquivos. Exemplo: chmod 755 arquivo.txt.
